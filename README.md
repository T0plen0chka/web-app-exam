# Веб-приложение «Электронная библиотека»

Информационная система для управления каталогом книг, просмотра детальной информации и ведения пользовательских рецензий. Проект разработан в рамках экзаменационного задания.

## Стек технологий
* **Backend:** Python 3.10+ / Flask
* **Database:** SQLite (SQLAlchemy ORM)
* **Frontend:** Bootstrap 5, HTML5, CSS3 (Кастомная адаптивная стилизация)
* **Аутентификация:** Flask-Login
* **Дополнительно:** Bleach (санация HTML), Markdown / EasyMDE (редактор рецензий)

---

## Основной функционал

1. **Главная страница (Каталог):**
   * Отображение книг в виде современных карточек («плитка») с обложками.
   * Полнотекстовый регистронезависимый поиск по названию и автору (с полной поддержкой кириллицы).
   * Фильтрация по годам издания, объёму страниц (от/до) и множественный выбор жанров.
   * Динамический подсчет средней оценки книги и количества рецензий.
   * Постраничная навигация (Пагинация).

2. **Интерактивные рецензии:**
   * Возможность авторизованным пользователям оставлять отзывы с оценкой от 0 до 5.
   * Поддержка форматирования Markdown при написании рецензии с использованием встроенного редактора EasyMDE.

3. **Ролевая модель доступа:**
   * **Администратор** (`Скрынникова Полина Андреевна`): Полный доступ к системе (добавление, редактирование, удаление книг, управление обложками).
   * **Модератор:** Возможность редактирования информации о книгах.
   * **Пользователь:** Просмотр книг и написание рецензий.

## Локальное развертывание

1. **Клонируйте репозиторий.**

2. **Создайте и активируйте виртуальное окружение:**
python -m venv library-env

Для Windows (PowerShell):
.\library-env\Scripts\Activate.ps1

Для Linux/macOS:
source library-env/bin/activate

3. **Установите зависимости:**
pip install -r requirements.txt

4. **Инициализируйте базу данных:**
flask init-db

Скрипт автоматически создаст таблицы, роли и тестовый аккаунт администратора:
Логин: admin | Пароль: password

(Данные для входа с правами модератора: Логин: moder | Пароль: password)

(Данные для входа с правами обычного пользователя: Логин: user | Пароль: password)

5. **Запустите сервер разработки:**
flask run

Приложение будет доступно по адресу: http://127.0.0.1:5000/
